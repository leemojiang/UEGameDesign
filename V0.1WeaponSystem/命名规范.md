# **变量（Properties）**
### **命名规则：小写开头驼峰（lowerCamelCase）**

```cpp
currentAmmo
maxAmmo
cooldownTime
spreadRecoveryRate
recoilStrength
```

### **为什么？**
- 与函数区分（函数大写开头）  
- 与类名区分（类名 PascalCase）  
- 蓝图显示自动格式化为 “Current Ammo”  
- UE 官方推荐风格  

### **反例（不要这样写）**
```cpp
CurrentAmmo
MAX_AMMO
spread_rate
```

---

# **布尔变量（Boolean Properties）**
### **命名规则：以 b 开头 + PascalCase**

```cpp
bIsFiring
bIsReloading
bHasAmmo
bIsCoolingDown
bUseStabilizer
```

### **为什么？**
- UE 官方布尔前缀规范  
- 蓝图显示为 “Is Firing”  
- 一眼能看出是 bool  

### **反例**
```cpp
isFiring
firing
IsReloading
```

# **函数（Functions）**
### **命名规则：大写开头驼峰（PascalCase）**

```cpp
Fire()
Reload()
IncreaseSpread()
ApplyRecoil()
RecoverSpread()
```

### **动词开头，表达行为**

### **反例**
```cpp
fire()
reload_weapon()
apply_recoil()
```

# **类名（Classes）**
### **命名规则：PascalCase + UE 前缀**

| 类型 | 前缀 | 示例 |
|------|------|------|
| UObject | U | UFireComp |
| Actor | A | AGenericFireArm |
| Component | U | UAmmoComp |
| Struct | F | FWeaponConfig |
| Enum | E | EFireMode |
| Interface | I | IFireable |

### **反例**
```cpp
FireComponent
genericFireArm
weapon_component
```

---

# **事件（Delegates / Events）**
### **命名规则：PascalCase + On 前缀**

```cpp
OnFire
OnFireFailed
OnAmmoChanged
OnSpreadChanged
OnRecoilApplied
```

### **事件处理函数（Handler）**
```cpp
OnFireHandle()
OnFireFailedHandle()
```

### **反例**
```cpp
fireEvent
fire_handler
eventOnFire
```

---

# **枚举（Enums）**
### **命名规则：E + PascalCase**

```cpp
enum class EFireMode
{
    Single,
    Burst,
    Auto
};
```

### **枚举值：PascalCase**
```cpp
EFireMode::Single
```

---

# **常量（Constants）**
### **命名规则：PascalCase + 明确含义**

```cpp
DefaultSpread
MaxRecoil
MinPitchLimit
```

### **反例**
```cpp
DEFAULT_SPREAD
max_recoil
```

# **最佳实践总结**

### ✔ 属性：小写开头驼峰  
### ✔ 布尔变量：b 前缀  
### ✔ 函数：PascalCase  
### ✔ 类名：PascalCase + UE 前缀  
### ✔ 事件：On + PascalCase  
### ✔ 枚举：E + PascalCase  
### ✔ 蓝图类：BP_ 前缀  
### ✔ 不使用下划线命名变量  
### ✔ 不使用全大写（除非宏）
